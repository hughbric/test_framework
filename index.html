Testing Framework

The Challenge

The aim of this exercise is to further explore JavaScript by implementing something found in every unit testing tool - an "assertEquals" method.

Expected Result

The following tests should "fail": 02, 03, 04, 07, 08 and 09
The following output is what to be expected for the initial couple of tests:
  • Test 02: Expected "abcdef" found "abc"
  • Test 03: Expected type Array but found Object
  • Test 04: Expected array length 2 but found 3
  • Test 07: Expected propB.propA[1].propB "b" but found "c"
  • Test 08: Expected propB.propC but was not found
  • Test 09: Expected type null but found type Object
